# Ramayana Hymns Repository

A comprehensive collection of hymns (stotras) and verses from Valmiki's Ramayana.

## About
This repository preserves and shares the beautiful hymns and spiritual teachings from Valmiki's Ramayana in Sanskrit with transliterations and translations.

## Structure
```
ramayana-hymns/
├── hymns/
│   ├── bala-kanda/          # Book 1: Childhood
│   ├── ayodhya-kanda/       # Book 2: Ayodhya
│   ├── aranya-kanda/        # Book 3: Forest
│   ├── kishkinda-kanda/     # Book 4: Kishkinda
│   ├── sundara-kanda/       # Book 5: Beautiful
│   ├── yuddha-kanda/        # Book 6: War
│   └── uttara-kanda/        # Book 7: After
```

## Current Collection

### Bala Kanda
- **Sarga 4**: Gayatri Hymn Connection - The spiritual framework of Ramayana

## Features
- Sanskrit verses with Devanagari script
- Transliteration in Roman script
- English translations
- Context and commentary
- Spiritual significance explained

## The Gayatri Connection
The entire Ramayana of 24,000 verses is structured on the 24-letter Gayatri mantra, making it not just an epic but a spiritual hymn itself.
