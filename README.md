# Ramayana Hymns Repository

A comprehensive collection of hymns (stotras) from the Ramayana.

## About
This repository aims to preserve and share the beautiful hymns from Valmiki's Ramayana in Sanskrit with translations.

## Contents
- Hymns organized by Kanda (book)
- Sanskrit text with transliteration
- English translations
- Audio references (where available)
